# Mamatalk
## サイト概要
### サイトテーマ
育児の悩みや成長の喜びを共有し、親同士が支え合えるコミュニティサイト
​
### テーマを選んだ理由
私には現在９ヶ月の子供がいます。

日々の育児の中で、妻から「同じ境遇の人と共感できるだけで気持ちが楽になる」という話を聞きました。

育児に関する情報は多く存在しますが、実際に同じ月齢の子供を育てている親の体験談や悩みを共有できる場は、

精神的な支えになると考えています。

​
そこで、育児中の親が気軽に日記を投稿し、

同じ悩みや喜びを共有できるサービスを作りたいと思い、このテーマを選びました。

### ターゲットユーザ
* 同じ月齢の子供を持つ親と交流したい人
* 育児の悩みや体験を共有したい人
* 初めて育児をする人
​
### 主な利用シーン
* 子育て中の悩みや不安を誰かに聞いて欲しい時
* 同じ月齢の子供を持つ親の体験談を知りたい時
* 子供の成長記録を残したい時
* 育児の嬉しかった出来事を共有したい時
